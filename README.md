# mediabazar-authorization

Authentication and user management API built with Nitro framework.

## Features

- User registration and authentication
- JWT token-based sessions with refresh tokens
- Password reset functionality
- User metadata management
- Cookie-based authentication
- MongoDB integration with Mongoose

## API Documentation

See `swagger.json` for complete API documentation.

## Development

Look at the [nitro quick start](https://nitro.build/guide#quick-start) to learn more how to get started.

### Scripts

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm preview` - Preview production build
- `pnpm test:api` - Run API tests
- `pnpm lint` - Lint code
- `pnpm format` - Format code
